CSS Transition patterns

Fan of stack

Based on the Fan of stack pattern from UI Transitions (look into the orientation category)

Description

When the user hover over an stack of elements, the stack can spread out some images to preview the stack's content.

Working Demo

Working demo

Also hosted on jsfiddle

The HTML needed

An <ul> with a class we can refer (elements in my case)

<ul class="elements">
    <li></li>
    <li></li>
    <li></li>
</ul>

The CSS needed

  1. It's needed to add position:relative so the <li> are positioned related to it. You need also to set some padding and margin so the <ul> can fill some space.

    .elements{
    margin: 100px;
    padding: 40px;
    position: relative;
}

  2. Set a width and height for each <li> (in my example, i'm using padding instead of height so the content inside can be centered vertically), we use position:absolute so they stack.

    We must add the CSS transition here. Below this, there is some extra styling to make it look better ;).

    .elements li{
    position: absolute;
    padding: 20px 0;
    width: 60px;
    -moz-transition: all .25s ease-in-out;
    -webkit-transition: all .25s ease-in-out;
    transition: all .25s ease-in-out;
    background-color: #EEE;
    border: 2px solid #FFF;
    box-shadow: 0 0 5px #888;
    text-align: center;
}

  3. Now we have to select each of the <li> and translate them clockwise, starting by placing the first above and to the left:

    .elements:hover li:nth-child(1) {
    -moz-transform: rotate(-15deg) translate(-2.5em,-2.5em);
    -webkit-transform: rotate(-15deg) translate(-2.5em,-2.5em);
    transform: rotate(-15deg) translate(-2.5em,-2.5em);
}

    I’ve also added a rotate to make it feel more natural. I’ve apply this only to the first six elements, so the second element has to be translated just upside, without any rotation:

    .elements:hover li:nth-child(2n) {
    -moz-transform: translate(-0.5em,-3em);
    -webkit-transform: translate(-0.5em,-3em);
    transform: translate(-0.5em,-3em);
}

  4. To indicate this is a stack, we can rotate some of the elements a bit in the static instance:

        .elements li:nth-child(5) {
        -moz-transform: rotate(7deg);
        -webkit-transform: rotate(7deg);
        transform: rotate(7deg);
    }
