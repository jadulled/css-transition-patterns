CSS Transition patterns

Minimization: help button

Based on the Fan of stack pattern from UI Transitions (look into the orientation category)

Description

Una acción auxiliar, en un espacio reducido, se expande hasta ocupar toda la pantalla.

Working Demo

Working demo

Also hosted on jsfiddle

The HTML needed

  1. La accion auxiliar que usaremos sera un help button, un div con la clase "hashCircle" que es el que se va a expandir y se convertira en la nueva “ventana” con informacion. Este estara contenido en un div que oculte todo lo que sea mas grande que el, ese contenedor sera "totalContainer".

  2. Las transitions se van a hacer apartir de que el "totalContainer" va a tener la clase active cuando se haya hecho click en el help button o no tenga la clase, cuando este navegando normalmente.

  3. El help button, tiene como hermano un div con la informacion que se quiera mostrar, este div es "hashCircleInfo". 

<div class="totalCotainer">
    <div class="hashCircle">
        <h2 class="hashCircleLabel"> ?</h2>
    </div>
    <div class="hashCircleInfo">
        <h2>Help Info</h2>
        <p>But I can't give you this case, it don't belong to me. Besides, I've already been through too much shit this morning over this case to hand it over to your dumb ass.</p>
    </div>
</div>

The CSS needed

  1. El "totalContainer" necesitaria tener definidos los siguientes estilos:

    .totalCotainer{
    overflow: hidden;
    width: 300px; /*algun tamaño fijo */
    min-height: 400px;
    position: relative;
}

    tambien tiene otros estilos que pueden modificarse o sacarse:

    .totalCotainer{
    background:#fff;
    border: 3px solid #ddd;
    margin: 10px auto;
}

  2. El help button, que es el div con la clase "hashCircle" tiene estilos necesarios para que se apliquen las transitions y para ubicarlo. Tambien se le define al elemento el tiempo de duracion y el ease-in que hace mas natural la transition:

    .hashCircle{
    display: block;
    cursor:pointer;
    position:absolute;
    bottom: 14px;
    right: 23px;
    transition: 1s ease-in;
    -moz-transition: 1s ease-in; /* Firefox 4 */
    -webkit-transition: 1s ease-in; /* Safari and Chrome */
    -o-transition: 1s ease-in; /* Opera */
}

    Tiene tambien estilos de formato, que son modificables:

    .hashCircle{
    background: none repeat scroll 0 0 #5ECDEB;
    border-radius: 1300px 1300px 1300px 1300px;
    -webkit-border-radius: 1300px 1300px 1300px 1300px;
    -moz-border-radius: 1300px 1300px 1300px 1300px;
}

    Definimos tambien el css para el estado activo, es decir, cuando el "totalContainer" tiene la clase "active" (se agrega una vez hecho click en el help button). En esta instancia lo que necesitamos es que el boton se expanda ocupando toda la pantalla, lo que usamos entonces es una transition de escala:

    .totalCotainer.active .hashCircle{
    -moz-transform:scaleX(80) scaleY(80);
    -webkit-transform:scaleX(80) scaleY(80);
}

  3. Estilo para el contenido del "hashCircle", en este caso es un <h2 class="hashCircleLabel"> ya que necesitamos que su opacidad cambie al expanderse, y necesitamos definir en que tiempo queremos que cambie esta opacidad, lo que es tambien una transition.

    Agregamos delay a esta transition para que el signo ? del help button no pise al contenido ("hashCircleInfo") cuando se hace visible. Este delay fue de 0.3s, este valor es el ultimo que se define.

    .hashCircle .hashCircleLabel{
    color: #FFFFFF;
    padding: 4px 11px;
    text-align: center;
    opacity:1;
    transition: 0.5s ease-in 0.3s;
    -moz-transition: 0.5s ease-in 0.3s; /* Firefox 4 */
    -webkit-transition: 0.5s ease-in 0.3s; /* Safari and Chrome */
    -o-transition: 0.5s ease-in 0.3s; /* Opera */
}

    y los estilos para cuando este activo el boton, cambian los valores de la transition y no tiene delay:

    .totalCotainer.active .hashCircle .hashCircleLabel{
    opacity:0;
    transition: 0.5s ease-in;
    -moz-transition: 0.5s ease-in; /* Firefox 4 */
    -webkit-transition: 0.5s ease-in; /* Safari and Chrome */
    -o-transition: 0.5s ease-in; /* Opera */
}

  4. El contenido del "hashCircle", que seria "hashCircleInfo" necesita no estar visible cuando no este activo y tiene definidos los estilos de su transicion de escala y de ubicacion, que deberia ser la misma que el help button o accion auxiliar ya que necesita seguir la transition del help button y asi parecer que salio de el.

    .hashCircleInfo{
    -moz-transform:scaleX(0.01) scaleY(0.01);
    -webkit-transform:scaleX(0.01) scaleY(0.01) translateY(30px);
    transition: 1s;
    -moz-transition: 1s; /* Firefox 4 */
    -webkit-transition: 1s; /* Safari and Chrome */
    -o-transition: 1s; /* Opera */
    opacity:0;
    position:absolute;
    bottom: 14px;
    right: 0;
}

    Por otro lado tiene css para darle formato que son estilos modificables:

    .hashCircleInfo{
    color:#fff;
    padding: 24px;
    font-size:13px;
}

    Por otro lado, se definen los estilos en su estado activo, en el que se define la transition de escala, de movimiento en el eje X y en el eje Y para ubicarlo dentro de la pagina, para asi agrandarse junto con el boton y aumentar su opacidad para ser visible:

    .totalCotainer.active .hashCircleInfo{
    -moz-transform:scaleX(1) scaleY(1) translateY(-190px);
    -webkit-transform:scaleX(1) scaleY(1) translateY(-190px);
    opacity:1;
    display:block;
    z-index: 6;
}

Javascript

Para que este estado activo exista es necesario agregar js. Para esto necesitamos:

  1. Por un lado un script para que funcione jquery:

     <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.0/jquery.min.js"> </script>

  2. Y necesitamos crear la funcion que agrega la clase

    <script type="text/javascript">
    doThings= function(){
                    $('.totalCotainer').addClass('active'); 
              }
</script>

  3. y se la agregaremos al help button:

    <div class="hashCircle" onclick="doThings()">

Casi terminado: opcion volver

Por ultimo se agrega dentro de "totalContainer" un link, "back", que nos permita salir de la ventana del help button, se agrega con js inline, que saca el estilo active al hacer click y asi salir del help button: 

    <a class="back" href="#" onclick="$('.totalCotainer').removeClass('active');">volver</a>

Cuando no se esta en la ventana de help button (osea sin la clase active) el boton de "back" no tiene que estar visible y cuando esta en active, aparece. Tambien tiene un estado hover que es modificable.

a.back{
    background: url("http://www.pedaling.com/images/go-back-icon.png") no-repeat scroll center center #ffffff;
    height: 30px;
    position: absolute;
    right: 0;
    text-indent: -9999px;
    top: 0;
    margin:6px;
    width: 30px;
    z-index: 7;
    display:none;
}
a.back:hover{  background-position: 2px 0;}
.totalCotainer.active  a.back{display:block}
