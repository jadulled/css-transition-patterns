Deal In

Live example

Tutorial

The idea is using the pattern Saving indicator from http://www.ui-transitions.com/#home (category Orientation Transitions) as reference but in reverse mode, make a dealing effect to a canvas.

We are using the animation tools from WestCiv that allows to change the animation properties in real-time with graphical interface. Making the task a lot easier

With the Daneden html code we trigger the animations (by jquery changes the button style when clicked)

  1. The html code:

    Head Section:

    <link rel="stylesheet" href="stylesheets/style.css">
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js"></script>

    2. Body:

    <div id="animateTest">
    <p>Test Card.</p>
    </div>
    <a class="butt butt-small" href="#" onclick="$('#animateTest').addClass('dealIn');">dealIn</a>

  2. Now the CSS.

    In css are two two things to define: First, give a name to the animation and use the global properties like animation name, duration, delay, iteration count, type of easing, and direction

    Let see:


    3. .dealIn { /*W3C*/ animation-name: dealIn; /*use this name to call the animation*/
    animation-duration: 1s ease-in-out;
    animation-delay: 0s;
    animation-iteration-count: 1; /*can be a number or infinite*/
    animation-timing-function: ease-in-out;
    animation-direction: normal; /* if you want to “rewind” use the option alternate.*/
    /*For webkit*/
    -webkit-animation-name: dealIn;
    -webkit-animation-duration: 1s;
    -webkit-animation-delay: 0s; -webkit-animation-iteration-count: infinite;
    -webkit-animation-timing-function: ease-in-out;
    -webkit-animation-direction: normal;
    /*For Mozilla*/
    -moz-animation-name: dealIn;
    -moz-animation-duration: 1s;
    -moz-animation-delay: 0s;
    -moz-animation-iteration-count: infinite;
    -moz-animation-timing-function: ease-in-out;
    -moz-animation-direction: normal;
    }

    Then, according to browser define the keyframes: They are instances where the properties will change over the animation.

    @keyframes dealIn {
    0% { transform: scale(0.6); transform-origin: 47% 53%;}
    25% {transform: scale(1.4) rotate(-56deg);transform-origin: 37% 53%;}
    100% {transform: scale(1.0) rotate(344deg) translateX(273px) translateY(205px);transform-origin: 50% 50%;}
    }
    @-webkit-keyframes dealIn {
    0% { -webkit-transform: scale(0.6); -webkit-transform-origin: 47% 53%; }
    25% {-webkit-transform: scale(1.4) rotate(-56deg); -webkit-transform-origin: 37% 53%; }
    100%{-webkit-transform: scale(1.0) rotate(344deg) translateX(273px) translateY(205px); -webkit-transform-origin: 50% 50%; }
    }
    @-moz-keyframes dealIn {
    0% { -moz-transform: scale(0.6); -moz-transform-origin: 47% 53%; }
    25% { -moz-transform: scale(1.4) rotate(-56deg); -moz-transform-origin: 37% 53%; }
    100% { -moz-transform: scale(1.0) rotate(304deg) translateX(273px) translateY(205px); -moz-transform-origin: 50% 50%; }
    }
